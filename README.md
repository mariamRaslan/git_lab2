# git_lab2

``` git branch -d <branch_Name>```
delete branch locally


``` git push origin :<branch_name> ```
 delete branch remotely

``` git tag ```
list all tags

``` git tag -d <tag_name> ```
delete tag locally

``` git push --delete origin <tagname> ```
delete tag remotely

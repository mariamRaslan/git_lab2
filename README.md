# git_lab2

``` git branch -d <branch_Name>```
to delete branch locally


``` git push origin :<branch_name> ```
to delete branch remotely

``` git tag ```
to list all tags

``` git tag -d <tag_name> ```
to delete tag locally

``` git push --delete origin <tagname> ```
to delete tag remotely
